- spout cam is not working :/ lets find a workaround
    
    Yeah, skip SpoutCam for now. Since **NDI Virtual Camera already shows up** in your MediaPipe dropdown, use **NDI as the workaround**.
    
    ## Workaround 1: TouchDesigner → NDI → MediaPipe
    
    In TouchDesigner, make this chain:
    
    ```
    videodevicein1 → resolution1 → null1 → ndiout1
    ```
    
    Add the node:
    
    ```
    NDI Out TOP
    ```
    
    Set `resolution1` to:
    
    ```
    1280 x 720
    ```
    
    Select `ndiout1` and set its name/source name to something like:
    
    ```
    kidist_td_cam
    ```
    
    Turn `Active` on.
    
    Then open the Windows Start menu and search:
    
    ```
    NDI Webcam Input
    ```
    
    or:
    
    ```
    NDI Launcher
    ```
    
    Open it. It may show up as a little icon in the Windows tray near the clock. Click it and select your TouchDesigner NDI source:
    
    ```
    kidist_td_cam
    ```
    
    Now go back to MediaPipe and choose:
    
    ```
    NDI Virtual Camera
    ```
    
    That is your fake webcam. It should now receive your TouchDesigner output.
    
    ## Workaround 2: Use OBS Virtual Camera
    
    Use this if NDI Webcam Input is annoying.
    
    In OBS:
    
    1. Add **Window Capture** or **Display Capture**
    2. Capture your TouchDesigner output or Perform Mode window
    3. Crop it if needed
    4. Click:
    
    ```
    Start Virtual Camera
    ```
    
    Then restart TouchDesigner and check MediaPipe’s webcam dropdown for:
    
    ```
    OBS Virtual Camera
    ```
    
    Select that in MediaPipe.
    
    This is less clean than NDI, but it usually works.
    
    ## Workaround 3: Just use webcam directly for now
    
    Since your webcam already works, choose:
    
    ```
    FaceTime HD Camera
    ```
    
    inside MediaPipe.
    
    Then turn on only one thing first:
    
    ```
    Detect poses: On
    Show overlays: On
    ```
    
    Click the matching:
    
    ```
    Generate GUI
    ```
    
    or pulse button.
    
    For your immediate goal, I’d do **NDI Virtual Camera** because it is already appearing in the dropdown. SpoutCam is the problem, not your camera or MediaPipe.
