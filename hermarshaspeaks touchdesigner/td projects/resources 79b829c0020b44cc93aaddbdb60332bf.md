    Below is a list of resources covering foundational concepts from computer graphics, creative coding and TouchDesigner usage. Covering a variety of format from textbook to interactive website, these resources were critical to my own learning and development, and I hope will be helpful to you as well!
    
    ### Computer Graphics Fundamentals
    
    [Texturing and Modeling, a Procedural Approach](http://elibrary.lt/resursai/Leidiniai/Litfund/Lithfund_leidiniai/IT/Texturing.and.Modeling.-.A.Procedural.Approach.3rd.edition.eBook-LRN.pdf)
    
    - Textbook written by some pioneers in computer graphics such as Perlin, Worley, and Peachey
    
    [Mathematics for 3D Game Design](https://canvas.projekti.info/ebooks/Mathematics%20for%203D%20Game%20Programming%20and%20Computer%20Graphics,%20Third%20Edition.pdf)
    
    - Written by the creator of Unreal Engine
    - Best introduction to computer graphics concepts like rendering pipeline, representing geometry quantitatively, and implementing commonly used algorithms
    - Also a great introduction / refresher for some computer-graphics specific math concepts like basic linear algebra, polar coordinates, etc
    
    [The Nature of Code](https://natureofcode.com/)
    
    - Free ebook teaching basic creative coding concepts
    - Is based in the Processing language, but is really helpful to build and intuition for how to translate nebulous concepts to concrete code
    
    ### Python Resources
    
    [Python in TouchDesigner](https://matthewragan.com/teaching-resources/touchdesigner/python-in-touchdesigner/) by Matthew Ragan
    
    - Comprehensive introduction to Python specifically for TouchDesigner users
    - The best place to start for Python beginners who plan to use in TouchDesigner
    
    [TouchDesigner | Understanding Extensions](https://matthewragan.com/2015/07/05/touchdesigner-understanding-extensions/)
    
    - A dedicated post by Matthew Ragan discussing extensions in TD and how they are used to scale up functionality quickly
    
    [Learn Python the Hard Way](https://learnpythonthehardway.org/book/)
    
    - In depth, very high quality book for Python beginners. Not specific to TouchDesigner or creative coding, so you will learn a ton of “real” python as well
    
    [Python Classes and Modules](https://docs.derivative.ca/Python_Classes_and_Modules)
    
    - TouchDesigner Python Documentation
    
    [Anaconda - Managing Python Environments and 3rd-Party Libraries in TouchDesigner](https://derivative.ca/community-post/tutorial/anaconda-managing-python-environments-and-3rd-party-libraries-touchdesigner)
    
    ### GLSL Resources
    
    [The Book of Shaders](https://thebookofshaders.com/)
    
    - The best introduction to fragment shaders. Interactive web-based book that will teach you the conceptual basics with real-time feedback
    
    [Inigo Quilez](https://iquilezles.org/articles/) Website
    
    - IQ is a master. This website is a collection of his blog posts and tutorials about fragment shader coding. Some are basic, some are quite advanced
